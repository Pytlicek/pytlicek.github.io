Configuration Instructions

Use the following settings for configuring your LibreSDR:

IP: 192.168.33.10
Username: root
Password: analog

fw_setenv ad936x_ext_refclk "<10000000>"
fw_setenv refclk_source external
fw_setenv xo_correction 10000000

fw_setenv ipaddr_eth "192.168.33.10"
pluto_reboot reset

Ensure all commands are entered correctly for proper configuration.
